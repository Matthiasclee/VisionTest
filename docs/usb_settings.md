# USB Settings

### Configurations List
* `vtcalib` presence: enabled calibration settings
* `run_commands.vt`: list of VTAC commands to run on the machine
* `VisionTest`: Git repository of VisionTest which can be pulled from using `usbupdate`
* `alternate_update_source_permitted` presence: allows `usbupdate` to pull from a nonstandard source
* `rshell_permitted` presence: allows reverse shells through the `rshell` command
